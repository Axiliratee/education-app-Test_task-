## Страница добавления образования

Проект создан для демонстрации страницы добавления образования с:

- слайдером документов,
- модальным окном с формой (React Hook Form + Zod),
- хранением данных в Zustand с persist (localStorage),
- адаптивной вёрсткой до 375px.

### Запуск

```bash
npm install
npm run dev
```
